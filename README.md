# MeetPlanner

## Описание идеи: ##
Приложение для назначения различных встреч.
### Возможности: ###
* Регистрация
* Аутентификация
* Просмотр всех встреч
* Добавление встречи
* Просмотр подробностей встречи
## Технические требования: ##
* Kotlin
* Python
* Django
* PostgreSQL
